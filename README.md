## Project Overview
This project analyzes Uber ride data to understand ride usage patterns across
categories, purposes, months, days, and time of day. The analysis is visualized
using Power BI to support data-driven decision making.

## Dataset
Source: Uber Dataset (CSV provided by faculty)
The dataset was cleaned and prepared in Excel before loading into Power BI.

## Data Preparation
- Removed duplicates
- Handled missing values
- Converted date columns (START_DATE, END_DATE)
- Created new columns such as Month, Year, and Trip Duration

## Dashboard Features
- Total Trips overview
- Trips by Category and Purpose
- Monthly and Daily ride trends
- Peak hour analysis
- Distance and duration insights

## Key Business Insights
- Ride demand peaks during specific months and weekdays
- Business rides contribute higher mileage
- Peak hours require better driver allocation
- Purpose-based insights help target marketing campaigns

## Business Recommendations
- Optimize ride allocation during peak months and hours
- Run purpose-based and seasonal promotions
- Improve service quality for high-demand categories
- Monitor long-duration and high-mileage trips

- Power BI
- Microsoft Excel
